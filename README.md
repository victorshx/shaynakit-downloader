# intro
a simple scraper to download all Figma design files from ShaynaKit ;)

`
Note: Paid account is required!
`

# getting started

## install
```
git clone https://github.com/victorshx/shaynakit-downloader
yarn
```

## run
To run the downloader, you need to enter your email and password in a .env.sample and rename it to .env
```
yarn start
```
